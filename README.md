## Các tính năng
- Quản lý xe ra vào bãi theo từng loại xe (xe máy, xe ô tô, xe khách)
- Thay đổi kích thước bãi xe, bảng giá vé
- Quản lý vé tháng
- Thống kê doanh thu
- Gửi mail khôi phục để lấy lại mật khẩu
- Quản lý thông tin các tài khoản (Admin)
